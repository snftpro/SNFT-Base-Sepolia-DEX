# SNFT-Base-Sepolia-DEX
Simple swap to exchange any EVM NativeToken to any ERC20 token you want



-Deploy DEX Contract:

https://thirdweb.com/0x10c9ce27B1C1Ff485A5EcA6ddC1AE5C462efCB2A/DEX/1.0.0/deploy
